# BPD dashboard

Dashboard locale per simulare il processo Camunda e controllare servizi, ordini e log.

## Avvio

```powershell
cd .\frotnend
npm start
```

Apri poi:

```text
http://127.0.0.1:5174
```

La dashboard usa gli script `start.ps1` e `stop.ps1` del progetto quando premi `Start stack` o `Stop stack`.
